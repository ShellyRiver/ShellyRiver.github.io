Room-Scale Pet Localization System

IoT / Edge + Web Localization

Real-time Room-level Localization

Developed a real-time room-level localization system using BLE RSSI via ESP32 receivers, a Raspberry Pi backend, and a lightweight web dashboard. It employs staggered receiver reporting to reduce Wi-Fi contention, and RSSI averaging combined with HMM forward inference for robust room prediction. The web UI shows live location and time distribution.

Accuracy: 87.7% → 90.9% HMM temporal smoothing
