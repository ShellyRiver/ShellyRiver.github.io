Reliable, Congestion-Aware Transport

Built a reliable, congestion-aware transport protocol over UDP with features for loss recovery, ordering, and fairness under contention. Implemented sender/receiver logic with sequence numbers, ACKs, retransmissions, buffers, and congestion control. Evaluated using Linux `tc` emulation for loss, delay, and bandwidth.
