Obstacle-Aware Multi-Agent Policy

Built an obstacle-aware multi-agent policy for dynamic UAV repositioning to maximize ground-user coverage while minimizing travel cost. The system uses a centralized training, decentralized execution model. It encodes obstacle elevation, user distribution, and "staleness" maps via a CNN, and uses attention-based neighbor aggregation for coordination without explicit role assignment.
